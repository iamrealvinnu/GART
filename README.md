# Dishana's Kitchen (GART)

A web application for ordering homemade food. This project allows users to browse a menu, manage a shopping cart, book delivery/pickup slots, and make payments. It also includes an admin panel for order management.

---
**Made by GDI Nexus Software Solutions**

## Features

- **Menu**: Browse available food items.
- **Cart**: Add and remove items, view total price.
- **Authentication**: User registration and login.
- **Slot Selection**: Choose a time slot for orders.
- **Profile**: Manage user information and view order history.
- **Admin Panel**: Manage the menu and track customer orders.
- **Responsive Design**: Works on mobile and desktop devices.

## Getting Started

### Prerequisites

- Node.js installed on your machine.
- npm (Node Package Manager).

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```

### Building for Production

To create a production build:
```bash
npm run build
```
The output will be in the `frontend/dist` folder.

## Deployment

The project is configured for deployment on Netlify. It uses a `netlify.toml` file to handle building and routing.
